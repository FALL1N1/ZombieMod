The first part is for server operators that install the plugin and do configuration. All settings, files and commands in Zombie:Reloaded is explained in detail.

The last part is for players. It explains various concepts and how to use the plugin's menus and its commands.

Introduction

About Zombie:Reloaded

A zombie trying to attack humans in zm_roy_the_ship.

Zombie:Reloaded is a SourceMod plugin made for Counter-Strike: Source where humans and zombies fight for elimination of eachother. It's an open source clone of ZombieMod with lots of additional features.

The game objectives will be changed into humans (counter-terrorists) versus zombies (terrorists). Humans try to survive by camping, making barricades and hiding. Zombies try to hit humans with their knife.

To balance the game, humans and zombies have different skills defined in player classes.

Common humans

  • have huge amounts of ammo
  • have bullets and explosions that push zombies away (knock back)

Common zombies

  • have no weapons (only their knife)
  • have much higher health
  • run faster
  • jump higher or longer
Humans fighting zombies on a bridge in zm_hellarch.

Initially there are no zombies. This gives humans some time to prepare themself. After a random time the round has started, one or more humans are turned into zombies.

There are three game modes in this game.

Classic

The classic game mode is just based on hiding, camping and barricading. It works on any map, but zombie maps (prefixed with "zm_") should be used for optimal balance and fun.

Escape

A second game mode is a escape from zombies to a safe place. Humans has to run through various obstacles and reach a safe place within a certain amount of time. Those who are left behind dies when the time is up. Escape maps are required for this mode. These maps are commonly prefixed by "ze_".

Riot

The third game mode is a defense against a zombie riot (based on the Zombie Riot plugin by Greyscale). Bots are usually used as zombies in this game mode but players may join the zombie team too. This game mode doesn't require special maps, but get best results if it's played with zombie maps (prefixed with "zm_" or "zr_").

In addition to the bots, each round is separated into days with different difficulty level. Humans advance a day by surviving a round. If they loose they will go back one day. Each day is a bit more difficult than the last one, where there are more zombies to kill.
